# Kisan Saathi

Kisan Saathi is an AI-powered agricultural advisory platform designed to help farmers access crop recommendations, agricultural information and multilingual support services.

## Features

- User Registration and Login
- JWT Authentication
- Role-Based Access Control
- RESTful APIs
- MySQL Database Integration
- Secure Backend Architecture
- Multilingual Support
- Localized Hindi & English Voice Assistant for rural accessibility
- Real-time speech recognition and voice-command processing using Web Speech API

## Tech Stack

- Java
- Spring Boot
- Spring Security
- JWT
- MySQL
- REST APIs
- HTML
- CSS
- JavaScript
-  Web Speech API (Voice Engine)

## Project Structure

- Backend: Spring Boot
- Database: MySQL
- Authentication: JWT
